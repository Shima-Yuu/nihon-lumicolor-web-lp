# 整体LP

サービス説明用のLP。

## URL

https://seitai-lp-65zo.vercel.app/

## 開発手順

### 前提

- Node.js 18.17以上
- npm、yarn または pnpm

### 1.パッケージのインストール

```bash
# 依存関係のインストール
npm install
# または
yarn install
# または
pnpm install
```

### 2.サーバーの起動

```bash
npm run dev
# または
yarn dev
# または
pnpm dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてアプリケーションを確認できます。

## コマンドまとめ

- `npm run dev` - 開発サーバーを起動
- `npm run build` - 本番用ビルドを作成
- `npm run start` - 本番用サーバーを起動
- `npm run lint` - コードリントを実行
